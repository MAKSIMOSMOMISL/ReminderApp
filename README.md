# 📋 Напоминания - Reminder App

Минималистичное приложение для напоминаний на C++ с использованием Qt6.

## 🚀 Возможности

- ✅ Добавление напоминаний с названием, описанием и датой/временем
- ✅ Удаление напоминаний
- ✅ Автоматическое уведомление при наступлении события
- ✅ Сворачивание полноэкранных приложений при срабатывании
- ✅ Работа в системном трее
- ✅ Сохранение напоминаний в JSON файл
- ✅ Кастомная иконка
- ✅ Темная минималистичная тема

## 🛠️ Технологии

- **C++17**
- **Qt6** (Core, Widgets, Svg)
- **CMake** (сборка)
- **MinGW** (компилятор)

## 📦 Установка и сборка

### Требования

- Qt6 (6.5.0 или новее)
- CMake 3.16+
- MinGW или MSVC

### Сборка

```bash
# Клонируйте репозиторий
git clone https://github.com/MAKSIMOSMOMISL/ReminderApp.git
cd ReminderApp

# Создайте папку сборки
mkdir build
cd build

# Настройка CMake (укажите путь к Qt)
cmake .. -DCMAKE_PREFIX_PATH="C:/Qt/6.11.1/mingw_64" -G "MinGW Makefiles"

# Сборка
mingw32-make
